gmacario.github.io
==================

Gianpaolo Macario public website on GitHub.com.

Automatically published to http://gmacario.github.io/

To test the website locally, install Jekyll then run

```
$ jekyll serve
```

If you want to display the draft pages as well:

```
$ jekyll serve --draft
```
